An error occured!

The following error was detected:

  • %error-status%: %error-message%


Now will follow details about the error and server.

  • Requested path: %path%
  • Server name: %server-name%
  • Server version: %server-version%


What you can try:

  • If the error ranges between 400 and 500: check your request and try again.
  • If the error is 5xx: As user, you are out of luck, this is a server error.
  • For admins/developers: The error goes as following: %error-message%, you can try to go to the log for more information.
  • For developers: If the message is 'Command not found', it means you have not properly registered the processor, make sure you are assigning it to the right server instance.


Developers: Configuring this page:

  • If you are a developer of this server and have access to the RaTs! Memory, you can rewrite this page.
  • To do so, you need to assign the 'connective.http.gen.error.provider' memory entry before the server starts, best way to do so is a coremodule.
  • The 'connective.http.gen.error.provider' entry is a BiFunction returning a String, the parameters are the HttpResponse object and the HttpRequest object.
